📄 Introdução ao HTML

HyperText Markup Language - A linguagem fundamental da web.

❓

O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas web. Documentos HTML podem ser interpretados por navegadores.

Criado por Tim Berners-Lee em 1991

⭐

Importância

  • Estrutura todas as páginas web
  • Base para CSS e JavaScript
  • Essencial para SEO
  • Acessibilidade web
  • Compatibilidade universal
🏗️

Estrutura Básica

<!DOCTYPE html>
<html>
    <head>
        <title>Título</title>
    </head>
    <body>
        Conteúdo
    </body>
</html>

🎨 Introdução ao CSS

Cascading Style Sheets - Estilo e beleza na web.

🎯

O que é CSS?

CSS é uma linguagem de estilo usada para descrever a apresentação de documentos HTML.

📦

Tipos de CSS

  • Inline: No elemento
  • Interno: Tag <style>
  • Externo: Arquivo .css
