📄 Introdução ao HTML

HyperText Markup Language - A linguagem fundamental da web.

❓

O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas web. Documentos HTML podem ser interpretados por navegadores.

Criado por Tim Berners-Lee em 1991

⭐

Importância

  • Estrutura todas as páginas web
  • Base para CSS e JavaScript
  • Essencial para SEO
  • Acessibilidade web
  • Compatibilidade universal
🏗️

Estrutura Básica

<!DOCTYPE html>
<html>
    <head>
        <title>Título</title>
    </head>
    <body>
        Conteúdo
    </body>
</html>

🎯 Introdução ao CSS

Cascading Style Sheets - Estilo e design para a web.

❓

O que é CSS?

CSS é a linguagem utilizada para estilizar e fazer o layout de páginas web. Com CSS podemos controlar cores, fontes, espaçamento e muito mais.

Especificação W3C desde 1996

⭐

3 Formas de Usar

  • Inline: Diretamente no elemento
  • Interno: Tag <style> no HTML
  • Externo: Arquivo .css separado
🎨

Sintaxe Básica

seletor {
    propriedade: valor;
    outra-prop: valor;
}

✏️ Seletores CSS

Aprenda a selecionar elementos para aplicar estilos.

📋 Tipos de Seletores
Tipo Sintaxe Exemplo Descrição
Elemento p {} p { color: blue; } Seleciona todos os parágrafos
Classe .nome {} .texto { font-size: 16px; } Seleciona por classe atribuída
ID #nome {} #header { background: gray; } Seleciona elemento único
Atributo [atributo] {} input[type="text"] {} Seleciona por atributo
Combinador a + b div > p { color: red; } Seleciona relações entre elementos

📦 Box Model CSS

Compreendendo margin, padding e border.

📐

O que é Box Model?

Todo elemento HTML é tratado como uma caixa (box). O box model descreve o espaço que ocupa na página.

Camadas:

  • Content (conteúdo)
  • Padding (espaço interno)
  • Border (borda)
  • Margin (espaço externo)
📏

Exemplo Visual

Elemento

Margin (20px) → Border (3px) → Padding (15px) → Content

🚀 Introdução ao JavaScript

A linguagem de programação da web - Tornando páginas interativas.

❓

O que é JavaScript?

JavaScript é uma linguagem de programação que torna as páginas web interativas. Ela pode ser executada no navegador do usuário.

Criada por Brendan Eich em 1995

⭐

Capacidades

  • Validar formulários
  • Criar animações
  • Manipular o DOM
  • Fazer requisições HTTP
  • Armazenar dados localmente
  • Criar aplicações web
💻

Seu Primeiro Script

<script>
    console.log("Olá, JavaScript!");
    alert("Bem-vindo!");
</script>

🔄 Operadores em JavaScript

Realização de operações matemáticas e lógicas.

➕

Operadores Aritméticos

  • + Adição: 5 + 3 = 8
  • - Subtração: 10 - 4 = 6
  • * Multiplicação: 7 * 2 = 14
  • / Divisão: 20 / 4 = 5
  • % Resto: 10 % 3 = 1
  • ** Potência: 2 ** 3 = 8
⚖️

Operadores de Comparação

  • == Igual
  • === Estritamente igual
  • != Diferente
  • !== Estritamente diferente
  • > Maior que
  • < Menor que
🎯

Operadores Lógicos

  • && AND (E)
  • || OR (OU)
  • ! NOT (NÃO)
  • Exemplo:
  • x > 5 && y < 10

🎯 Funções em JavaScript

Reutilizando código com funções.

📐

O que é Função?

Uma função é um bloco de código reutilizável que executa uma tarefa específica. Funções ajudam a organizar o código e evitar repetição.

Vantagens:

  • Reutilização de código
  • Código mais organizado
  • Facilita manutenção
💻

Sintaxe de Função

function saudacao(nome) {
    return "Olá, " + nome;
}

saudacao("João");
⚡

Arrow Function

Sintaxe moderna e mais concisa:

const soma = (a, b) => a + b;

soma(5, 3); // 8
